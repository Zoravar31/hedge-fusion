Architecture

Built like a real trading desk

Three teams working in sequence. Each agent has one job and hands its output to the next. No agent can be skipped.

Analyst team 4 agents · run in parallel
01
Fundamentals
P/E, balance sheet, DCF, promoter pledge, ROCE. Flags SEBI enforcement history.
02
Technical
200 DMA, RSI, MACD, Supertrend, Bollinger Bands. India circuit-breaker proximity check.
03
Sentiment
FII/DII flows, bulk/block deals, ET/Moneycontrol tone, promoter stake changes.
04
News
RBI/SEBI macro, sector policy, INR/USD impact on exporters, upcoming result dates.
↓
Research team 3 agents · bull vs bear debate
05
Bull Researcher
Builds the strongest buy case from analyst reports. Assigns conviction 0–1.
06
Bear Researcher
Builds the strongest sell/avoid case. Pre-empts bull arguments directly.
07
Research Manager
Adjudicates the debate. Issues BUY/SELL/HOLD with entry zone, stop, target. PASS or BLOCK.
↓
Execution team 3 agents · order → risk check → PM gate
08
Trader
Converts research verdict into exact Kite Connect order params: qty, price, SL, CNC/MIS.
09
Risk Manager
Validates position size, R:R ≥ 1:2, sector concentration <30%, SEBI compliance.
10
Portfolio Manager
Final APPROVE or VETO. All 6 criteria must pass. Only then does the order reach Zerodha.
Features

What makes it Indian-first

Not a US equity framework with Indian tickers pasted in. Every prompt, tool, and default is built around NSE, SEBI, and Zerodha.

🛡️
Paper mode by default
Every order is simulated and logged to CSV. Flip one line in .env to go live. No accidental trades.
⚔️
Bull vs Bear debate
Two dedicated researchers argue opposing cases before the Research Manager adjudicates. One-sided calls get caught here.
🚦
Portfolio Manager gate
Hard veto if R:R < 1:2, sector concentration >30%, or any SEBI/promoter red flag. The order never reaches Zerodha without this gate passing.
📡
India news feeds
RSS feeds from Economic Times, Moneycontrol, Business Standard, NSE corporate announcements, and RBI bulletins — live, no API key needed.
⚡
Parallel analysts
All 4 analysts run concurrently via ThreadPoolExecutor. Same cost, roughly 4× faster than running them sequentially.
🕘
Daily scheduler
Run scheduler.py and it fires every trading day at your set time — skipping weekends and NSE market holidays automatically.
Get started

Running in 5 minutes

One conda environment, one API key, one command.

# 1. Clone
git clone https://github.com/Zoravar31/hedge-fusion.git
cd hedge-fusion

# 2. Environment (Anaconda recommended on Windows)
conda create -n hedge_fusion python=3.11 -y
conda activate hedge_fusion
pip install -r requirements.txt

# 3. Configure
cp .env.example .env
# Edit .env — add your OPENAI_API_KEY
OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxx
MODEL_NAME=gpt-4o-mini
KITE_PAPER_TRADE=true
# Analysis only — no order placed
python main.py RELIANCE

# Full pipeline + paper trade if PM approves
python main.py HDFCBANK --execute
